## July 2026

### Platform on Astro + Kleap Database

**Faster sites, built-in database** ⚡

- Kleap now builds **Astro** websites (Tailwind CSS v4, React islands) — static-first for speed and SEO
- Sites publish **automatically** to `your-app.kleap.io` on a global edge network, with free SSL
- **Kleap Database** — an optional built-in database with user accounts — add it on demand for saved data and logins
- **Custom domains** connect via a simple A record with automatic SSL
- Built-in **Forms** collect submissions to your dashboard with email notifications

## Version Numbers

Kleap uses continuous deployment, so we don't have traditional version numbers. Updates are deployed as they're ready, and this changelog reflects significant changes.

## Breaking Changes

We strive to avoid breaking changes. When necessary, they are:

- Announced in advance
- Documented with migration guides
- Rolled out gradually
